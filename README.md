# Legal Redline Analyzer

Legal Redline Analyzer — локальное приложение на Python для сравнения действующей и предлагаемой редакции нормы законодательства, выявления юридически значимых изменений и подготовки предварительного обзора влияния на деятельность Общества.

Приложение работает полностью локально: документы, база SQLite, история анализов и экспортированные файлы остаются на компьютере пользователя. В приложении нет OpenAI API, Google API, облачных LLM, платных API, внешних серверов и отправки документов во внешние сервисы.

## Установка зависимостей

В папке проекта выполните команду:

    pip install -r requirements.txt

## Запуск

В папке проекта выполните команду:

    streamlit run app.py

## Первый вход администратора

При первом запуске автоматически создается администратор:

- login: admin
- password: admin123

После первого входа приложение предложит сменить пароль.

## Основные возможности

- Авторизация с ролями администратора и обычного пользователя.
- Сравнение действующей и предлагаемой редакции нормы через difflib.
- Анализ ключевых слов по категориям: обязанности, права, запреты, финансы, операционное влияние, полномочия госорганов, расширительное толкование, ответственность и инфраструктура.
- Предварительный юридический обзор влияния на деятельность Общества.
- Локальная библиотека законодательства для DOCX, PDF и TXT.
- Извлечение текста из DOCX, включая таблицы.
- Извлечение текста из PDF через pypdf с fallback через pdfplumber.
- История анализов с разграничением доступа.
- Редактируемые словари анализа для администратора.
- Экспорт результата в Word и Excel.

## Смена пароля

Откройте раздел Настройки, введите новый пароль и подтвердите его. Администратор также может сбрасывать пароль пользователей в разделе Администрирование.

## Загрузка документов

Раздел Библиотека законодательства позволяет администратору загрузить .docx, .pdf или .txt. Файл сохраняется локально в data/legal_library/, а сведения о нем записываются в SQLite.

Если PDF не содержит текстового слоя, приложение показывает предупреждение:

> Текст из PDF не извлечен. Вероятно, документ является сканированным. Для анализа требуется текстовый PDF, DOCX или TXT.

Облачный OCR не используется.

## Проведение анализа

Откройте Новый анализ, заполните действующую и предлагаемую редакции нормы, укажите название документа, статью/пункт, сферу регулирования и тип анализа. Нажмите Сравнить и сформировать юридический обзор.

Анализ использует difflib, локальные словари ключевых слов, правила юридической классификации и шаблонную генерацию обзора. Итог не является окончательной позицией Общества и требует проверки юристом.

## Экспорт Word и Excel

После анализа доступны кнопки экспорта. Обычный пользователь выгружает юридический обзор в Word. Администратор также может выгружать таблицы в Excel.

Экспортированные файлы сохраняются в data/exports/.

## Роли

Администратор управляет пользователями, библиотекой документов, словарями анализа и всей историей. Обычный пользователь запускает анализ, выгружает Word и видит только свои анализы.

## Локальность и безопасность

Пароли хранятся в хэшированном виде с использованием bcrypt. Приложение не выполняет сетевые API-вызовы и не отправляет документы во внешние сервисы. В конфигурации Streamlit отключен сбор статистики использования.
